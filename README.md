# 📱 React_Native_Practice 1.0

🚀 A modern React Native mobile application built using Expo.

This project demonstrates authentication, Facebook login integration, and a basic E-Shopping workflow with clean UI and structured navigation.

---

## 🧠 Project Overview

React_Native_Practice 1.0 is a cross-platform mobile application developed to practice real-world mobile app development concepts.

It includes:

- 🔐 Signup & Registration Form
- 📘 Facebook Login
- 🛍️ E-Shopping Product Screens
- 🛒 Add to Cart UI
- 📱 Clean Navigation Structure

The goal of this project is to strengthen React Native fundamentals and understand mobile authentication and commerce flows.

---

## ✨ Features

- ✅ Signup Page  
- ✅ Registration Form with Input Validation

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/React_Native_Practice-1.0.git
cd React_Native_Practice-1.0
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start the Application

```bash
npx expo start
```

You can run the app on:

- 📱 Android Emulator  
- 🍎 iOS Simulator  
- 📲 Expo Go  
- 🔧 Development Build
